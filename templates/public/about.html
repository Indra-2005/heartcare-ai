About HeartCare AI

An intelligent cardiovascular risk screening system built with machine learning, designed to support early detection and preventive care.

Project Overview

HeartCare AI uses a Random Forest classifier trained on the Cleveland Heart Disease dataset to predict cardiovascular risk from 13 clinical parameters. The goal is to provide individuals with actionable, data-driven heart health insights.


This project was developed as a full-stack machine learning web application combining Flask backend APIs with a modern, accessible frontend.

Machine Learning Model

  • Algorithm: Random Forest Classifier
  • Dataset: Cleveland Heart Disease (UCI ML Repository)
  • Features: 13 clinical attributes
  • Target: Binary (heart disease: yes/no)
  • Evaluation: ~98.5% accuracy on test split
  • Output: Class prediction + probability score

Clinical Parameters

The 13 features used by the model for prediction

Age
Biological Sex
